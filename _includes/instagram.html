New: Last week's post talked about the injection of the pcm.js script. Meta claimed they only inject the script to respect the user's ATT choice, and additional "security and user features".

After improving the JavaScript detection, I now found some additional commands Instagram executes:

  • Instagram iOS subscribes to every tap on any button, link, image or other component on external websites rendered inside the Instagram app.
  • Instagram iOS subscribes to every time the user focuses on a UI element (like a text field) on third party websites rendered inside the Instagram app.
